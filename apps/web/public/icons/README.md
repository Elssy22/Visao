# Icons PWA

Ce dossier doit contenir les icônes de l'application pour la PWA.

## Icônes requises

Générez ces icônes à partir de votre logo (recommandé: 512x512 PNG):

- `icon-16x16.png`
- `icon-32x32.png`
- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

### Pour iOS (Apple Touch Icon)
- `apple-touch-icon.png` (180x180)

### Pour les splash screens iOS (optionnel)
- `apple-splash-1125-2436.png` (iPhone X)
- `apple-splash-1170-2532.png` (iPhone 12/13)
- `apple-splash-2048-2732.png` (iPad Pro)

### Badge pour les notifications
- `badge-72x72.png` (monochrome, pour Android)

## Outil recommandé

Utilisez un générateur comme:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Uploadez votre logo 512x512 et téléchargez le pack d'icônes complet.
